# I2PChat v0.4.1 — UI update + profile/lock fixes

## Что вошло в релиз

### 1) Новый пользовательский интерфейс в стиле macOS для всех платформ

- Полностью обновлён визуальный стиль приложения под macOS-like дизайн.
- Добавлены две темы интерфейса: `ligth` и `night`.
- Темы работают на всех целевых платформах (Linux, macOS, Windows).
- Переключение тем перенесено в основное окно (быстрый переключатель рядом со строкой статуса).
- Улучшены элементы интерфейса: кнопки действий, компактные панели, popup-меню, иконки и статусная строка.

### 2) Фикс логики `Lock to peer` и сохранения профиля

- Исправлена проблема, при которой при первом `Lock to peer` в `.dat` мог сохраняться только адрес пира.
- Нормализован формат профиля `.dat`:
  - строка 1: приватный ключ профиля,
  - строка 2: `stored peer` (если профиль закреплён за собеседником).
- Исправлена загрузка профиля для ранее повреждённых/нестандартных `.dat`.
- Убраны дубли строк при повторном `Lock to peer`.
- После первого корректного lock профиль теперь подхватывает закреплённый peer без повторного ввода и без двойного перезапуска.

---

## Итог

`v0.4.1` стабилизирует UX и профильную логику:
- единый современный UI с двумя темами для всех платформ;
- корректная и предсказуемая работа `Lock to peer` с первого раза.
