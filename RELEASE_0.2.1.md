# I2PChat 0.2.1

Стабильный релиз с доработками передачи файлов и картинок, подтверждением доставки и улучшениями UI.

---

## Что нового

### Передача файлов и картинок
- **Поддержка картинок**: отправка и приём изображений (PNG, JPEG, WebP, GIF), кнопка «Send Pic», превью в чате
- **Подтверждение доставки файлов**: получатель отправляет FILE_ACK; у отправителя отображаются галочки (✓/✓✓) для файлов и картинок, как для сообщений
- **Отклонение файла**: при отказе получателя от приёма отправитель видит уведомление (REJECT_FILE)
- **Прогресс**: прогресс-бар при отправке и приёме файлов/картинок; прогресс каждые 4 KB при маленьких передачах; тень галочки на белых картинках для читаемости
- **Файлы**: убраны дубли «File sent/received»; в «File received» — ссылка «Открыть папку загрузок» по клику; в диалоге профиля — кликабельная папка профилей (как «Open downloads» в чате)
- **Отмена передачи**: синхронизация отмены (ABORT_FILE) между пирами; только PNG/JPEG для картинок в UI

### Надёжность и протокол
- **HMAC**: явная кодировка UTF-8 для кроссплатформенности; отключение при ошибке целостности; логи msg_type/body_len
- **Таймауты и отключения**: исправлено отключение по idle; отключение при потере соединения во время передачи; keepalive против таймаута; таймаут не срабатывает во время передачи файла
- **Подключение**: ожидание туннелей I2P перед «Online!» и запуском accept_loop; затем возврат к немедленному «Online!» и старту accept_loop без блокировки
- **Ядро**: парсинг первого сообщения как фрейма (S+len+body), не readline; одна активная receive_loop; сохранение Unicode-имён файлов (кириллица и др.)

### Интерфейс
- **Окно**: центрирование кнопок профиля, Cmd+Q и меню Quit, заголовок без версии, разделитель @
- **Тема**: тёмная тема для диалогов QMessageBox
- **Статусы**: понятные сообщения Wait / Ready при подключении; анимированный прогресс-бар в чате; сообщение «Starting I2P session» вместо «Building»

### Сборка и окружение
- **.gitignore**: добавлены build/, dist/, __pycache__/, .DS_Store, .cursor
- Сборка под **Python 3.14+**, PyInstaller; кросс-платформенные скрипты (macOS, Linux, Windows)

## Скачать

- **macOS (arm64)**: `I2PChat-macOS-arm64.zip` — распаковать и перенести I2PChat.app в /Applications
- **Windows (x64)**: `I2PChat-windows-x64.zip` — распаковать и запустить I2PChat\I2PChat.exe
- Другие платформы — см. [Releases](https://github.com/MetanoicArmor/I2PChat/releases)

## Требования

- [i2pd](https://i2pd.website) с включённым SAM (порт 7656 по умолчанию)
- Для сборки из исходников: Python 3.14+, зависимости из `requirements.txt`
